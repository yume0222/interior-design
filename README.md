# コーポレートサイト

## 制作背景
スクロール時のフェードインやタブ切り替え、モーダルウィンドウ、アコーディオンなどの実装を通して、JavaScriptによる基本的な動きの理解と再現力の習得を目的として制作しました。

---

## 使用技術
HTML, CSS, JavaScript

---

## 工夫した点
CSS設計を意識し、BEMやコンポーネント化、CSS変数の活用を行いました。<br><br>

レスポンシブ対応の際にレイアウトが崩れやすかったため、max-widthや%指定を用いて画面サイズに応じて柔軟に調整できるよう工夫しました。<br><br>

制作後もアクセシビリティや保守性について学習を続け、新たに得た知識をもとに既存のコードを見直しました。<br>
WAI-ARIAを活用したアクセシブルなハンバーガーメニューを実装しました。aria-expandedやaria-controlsなどをJavaScriptと連動させ、スクリーンリーダー利用者にも開閉状態が伝わるよう配慮しています。<br>
アクセシビリティとユーザビリティを意識し、ラジオボタン・チェックボックスをカスタマイズしながらもキーボード操作を維持できるフォームを実装を行いました。また、fieldset・legendによる適切なグループ化や、意味論に沿ったフォーム構造を採用しています。<br>
WAI-ARIAを活用し、スクリーンリーダーやキーボード操作に対応したアクセシブルなタブUIを実装しました。roleやaria-*属性を適切に設定し、JavaScriptで状態管理を行っています。<br>
コメントアウトを追加・整理し、可読性と保守性を向上させました。

---

また、意図しないタイミングで動作してしまう不具合の調整にも時間がかかりました。
